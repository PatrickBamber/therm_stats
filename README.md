# therm_stats

ThermStats is a Flutter-based Android application designed to fetch, display, and analyze temperature data from a predefined endpoint. The app showcases user temperatures, highlights individuals with elevated temperatures (above 37Â°C), and creatively presents age-related statistics. It aims to demonstrate data manipulation, network requests, and UI design using Flutter and Dart.
